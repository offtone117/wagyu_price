# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？

- リモートリポジトリはネットワーク上のリポジトリでローカルは個人の作業環境でのリポジトリ。

## プッシュとマージの違いは何でしょうか？

- プッシュはリモートとローカル間で変更履歴を取り込むことで、マージは別のブランチの作業内容をブランチに取り込むこと。

## コミットとプッシュの違い

- コミットは変更内容のお知らせ、プッシュは変更内容を共有して実際に取り込むこと。

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？

- 作業内容が分かるように書く。

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？

- プルリクエストはリモートでのマージになるし、レビューが入るため工数が増える。

## コンフリクトを起こしてしまった場合、どう対処すべきですか？

- 先にマージされた内容を取り込むか、後にマージされた内容を取り込むか、両方取り込む。
